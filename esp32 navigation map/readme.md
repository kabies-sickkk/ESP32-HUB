ESP32:

Nhận dữ liệu từ app điều hướng qua BLE.

Hiển thị hướng rẽ, tốc độ, khoảng cách trên màn hình OLED.

Cập nhật thanh tiến độ khi đang di chuyển.

Vẽ chữ "H" khi đã đến đích.
